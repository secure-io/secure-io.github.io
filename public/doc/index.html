Introduction

Secure I/O is an open-source project that provides provable secure authenticated encryption for data streams.

It has been developed because many widely used solutions for data encryption are not well-suited for streaming or cannot preserve important security guarantees.

Take a look at the FAQ to learn more about the limitations of classical encryption schemes.
